# Code of Conduct

## Our Standards

Phtps is a technical project. Everyone contributing — through issues, pull requests, discussions, or code review — is expected to:

- Be direct and constructive in feedback
- Respect that people have different levels of experience
- Focus disagreements on the technical idea, not the person
- Accept that maintainers make final decisions on scope and direction

The following are not acceptable:

- Personal attacks or insults
- Harassment of any kind
- Dismissing contributions without explanation

---

## Enforcement

If you experience or witness unacceptable behaviour, report it to: **conduct@your-domain.com**

Reports will be reviewed within 48 hours. Confirmed violations result in removal from the project.

---

## Scope

This code of conduct applies to all Phtps spaces — GitHub issues, pull requests, discussions, and any other official channels.
